# Beginner Free Contest 12 - DPD1

**Thể loại:** Dynamic Programming  
**Độ khó:** 800

## Đề bài

Cho 3 số nguyên dương L, R, k. Tính tổng các số thuộc đoạn [L, R] và có không quá k chữ số phân biệt.

## Dữ liệu

Gồm một dòng duy nhất chứa 3 số nguyên dương L, R, k (1 ≤ L ≤ R ≤ 10¹⁸, k ≤ 10).

## Kết quả

In ra kết quả bài toán khi chia lấy dư cho 998244353.

## Ví dụ

**Input:**
```
10 50 2
```

**Output:**
```
1230
```
