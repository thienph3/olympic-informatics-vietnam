# Beginner Free Contest 40 - BASIC

**Thể loại:** Brute Force  
**Độ khó:** 300

## Đề bài

Cho một mảng gồm N số nguyên a₁, a₂, ..., aₙ. Hãy tìm 3 số i, j, k sao cho ai ≠ aj ≠ ak và ai + aj + ak - ai ⊕ aj ⊕ ak đạt max.

## Dữ liệu

- Dòng đầu tiên gồm một số duy nhất N (3 ≤ N ≤ 10³).
- Dòng thứ hai gồm N số nguyên a₁, a₂, ..., aₙ (1 ≤ ai ≤ 10⁴).

## Kết quả

In ra một dòng duy nhất là giá trị lớn nhất cần tìm.

## Ví dụ

**Input:**
```
5
1 2 3 4 5
```

**Output:**
```
10
```
