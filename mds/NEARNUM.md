# Beginner Free Contest 39 - NEARNUM

**Thể loại:** Math  
**Độ khó:** 350

## Đề bài

Một hôm, Lộc mở vở bài tập lập trình lúc nhỏ ra xem. Lộc thấy một bài khá thú vị như sau:

Đề bài cho một số N. Bạn hãy in ra màn hình số M có tổng các chữ số chia hết cho 4 nhỏ nhất và lớn hơn hoặc bằng số N đã cho (N ≤ M và M nhỏ nhất).

## Dữ liệu

Dòng duy nhất chứa số nguyên N (1 ≤ N ≤ 10¹⁰⁰).

## Kết quả

In ra màn hình số M có tổng các chữ số chia hết cho 4 nhỏ nhất và lớn hơn hoặc bằng số N đã cho (N ≤ M và M nhỏ nhất).

## Ví dụ

**Input:**
```
432
```

**Output:**
```
435
```

**Input:**
```
99
```

**Output:**
```
103
```

**Input:**
```
237
```

**Output:**
```
237
```
